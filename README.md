# 🧛 Survivors Clone (Godot 4)

This is a personal learning project made with **Godot 4**, where I'm recreating the core gameplay mechanics of *Vampire Survivors* using GDScript.
It's still a work in progress and meant for practicing game development fundamentals.

## 🛠️ Technologies Used

- **Engine**: [Godot Engine 4](https://godotengine.org/)
- **Language**: [GDScript](https://docs.godotengine.org/en/stable/getting_started/scripting/gdscript/index.html)
